# Python-Projects

A collection of beginner-friendly Python projects built for practice, learning, and skill improvement.

This repository contains a mix of **Python scripts** and **Jupyter notebooks** covering core programming concepts such as:
- variables and data types
- loops and conditions
- functions
- random number generation
- user input handling
- simple game logic
- password generation
- basic project structure

---

## Projects Included

### 1. CAFE_MANAGMENT.py
A simple cafe management program where users can explore menu items and manage basic order-related operations.

### 2. COUNTDOWN_TIMER.ipynb
A countdown timer project that counts down from a given time and displays the remaining seconds.

### 3. GUESS_THE_NUMBER(COMPUTER).ipynb
A number guessing game where the computer guesses the number.

### 4. GUESS_THE_NUMBER.ipynb
A classic number guessing game where the user tries to guess the correct number.

### 5. MADLIBS.ipynb
A fun Mad Libs game that generates a story based on user input.

### 6. PASSWORD_GENERATOR.ipynb
A password generator project that creates strong and random passwords.

### 7. QUIZ_GAME.py
A simple quiz game with questions and score tracking.

### 8. ROCK_PAPER_SCISSORS.ipynb
A Rock, Paper, Scissors game built using Python logic and random choice.
